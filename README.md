# Expense Tracker Application

A Python-based Expense Tracker application used to manage daily expenses.
The application allows users to add expenses, view expense records, and calculate total spending.

## Technologies Used

- Python

## Features

- Add expenses
- View expense list
- Calculate total expenses
- Menu-driven console application

## Concepts Used

- Python Lists
- Dictionary
- Functions
- Loops
- Conditional Statements

## Project Purpose

This project helps to understand Python programming concepts and basic data management.

## How to Run

1. Install Python
2. Run the file:

python expense_tracker.py
